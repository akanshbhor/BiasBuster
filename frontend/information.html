The Problem We Solve

BiasBuster was engineered for CodeQuest 2026 to tackle a critical vulnerability in modern, large-scale AI deployments: the silent propagation of workplace bias. As AI assumes a foundational role in drafting performance reviews, job descriptions, and core corporate communications, it frequently encodes human biases—such as gendered expectations, ageism, classism, and ableism—into its outputs. These subtle biases are difficult to catch with simple string matching. BiasBuster acts as a proactive controller for algorithmic governance, identifying and explaining these deep contextual dog-whistles to ensure ethical alignment and systemic integrity in autonomous systems.

Our Multi-Tiered Intelligent Routing System

To achieve high-fidelity detection without compromising user experience, BiasBuster utilizes a unique hybrid approach to semantic evaluation:

  • Layer 1: High-Speed Heuristics: A lightning-fast, local NLP engine utilizes expanded synonym matching and direct database parsing (bias_database.csv) to instantly return explicit bias flags to the UI, providing immediate visual feedback in under 300ms.
  • Layer 2: Asynchronous Verification Daemon: A silent, backend Python daemon thread receives the text and executes a deep semantic audit. This process asynchronously calls a state-of-the-art Large Language Model (e.g., GPT-OSS 20B with automated failovers to Llama 3.3). It extracts complex contextual dog-whistles the local engine misses, categorize them with high/medium severity, and explains the impact on specific demographic groups.
  • Custom Deduplication Pipeline: A resilient frontend JavaScript pipeline isolates multi-layered text highlights from dynamic, deduplicated sidebar metrics, creating a seamless and logical user interface.
